# OPTIONAL - Apple Metal GPU Inference (macOS Apple Silicon only)
# ----------------------------------------------------------------------------
# Enables the Metal / MLX provider for on-device AI descriptions.
# Install this on macOS with Apple Silicon before building the app:
#   pip install mlx-vlm
# It is intentionally excluded from the default requirements because it is
# macOS-only and has many heavy dependencies (transformers, mlx, etc.).
# If not installed, the MLX provider will be silently disabled at runtime.
# 0.6.x is required to LOAD the Qwen3-VL models in the known-good list: 0.4.x
# fails at runtime with "Unrecognized video processor" on transformers 5.x.
# Verified working: mlx-vlm 0.6.3 + transformers 5.5.0 (older Qwen2.x/Gemma still load).
mlx-vlm>=0.6.3 ; sys_platform == 'darwin' and platform_machine == 'arm64'

# ----------------------------------------------------------------------------
# OPTIONAL - Advanced Features
# ----------------------------------------------------------------------------
# Required for MLX provider when using Phi-3.5-Vision or other models whose
# transformers processors request return_tensors="pt" internally. Inference
# still runs on Metal via MLX — torch is only used for the processor layer.
# Qwen2-VL / Qwen2.5-VL models work without torch.
# Note: llava-1.5-7b-4bit is NOT supported (breaks with transformers 5.x).
torch>=2.0.0 ; sys_platform == 'darwin' and platform_machine == 'arm64'        # CPU-only, needed by transformers processor
torchvision ; sys_platform == 'darwin' and platform_machine == 'arm64'         # required by torch
# transformers>=4.30.0 # Already pulled in by mlx-vlm
